> **Validate your startup idea in seconds using AI — completely free!**
> Get a full business analysis including SWOT, market opportunity, revenue models, risk analysis, and actionable next steps.

## 🌐 Live Demo
👉 **[Try the App Here](https://smart-startup-validator-a3dpvd5z4zpkubgpterfg4.streamlit.app/)**

## ✨ What Does This App Do?

You enter your startup idea and the AI gives you a complete validation report:

| Section | What You Get |
|---|---|
| 🔍 SWOT Analysis | Strengths, Weaknesses, Opportunities, Threats |
| 📈 Market Opportunity | Market size, pain points, competition overview |
| 💵 Revenue Models | 3 specific ways to make money from your idea |
| ⚠️ Risk Analysis | Top 3 risks with mitigation strategies |
| ✅ Next Steps | 3 actionable steps with timelines |
| 🏁 Overall Verdict | Honest 2-3 sentence summary of viability |

---

## 🎯 How To Use It

1. Open the **[Live App](https://smart-startup-validator-a3dpvd5z4zpkubgpterfg4.streamlit.app/)**
2. Get your **free API key** from [openrouter.ai](https://openrouter.ai) — takes 2 minutes, no credit card
3. Paste the key in the **sidebar**
4. Fill in your startup details
5. Click **Validate My Idea** and get your report in 15-30 seconds!

## 💻 Run It Locally

**Step 1 — Clone the repo**
git clone https://github.com/kollivibash/smart-startup-validator.git
cd smart-startup-validator

**Step 2 — Create virtual environment**
python3 -m venv venv
source venv/bin/activate

**Step 3 — Install dependencies**
pip install -r requirements.txt

**Step 4 — Get your free API key**
Go to openrouter.ai — Sign up — Keys — Create Key

**Step 5 — Run the app**
streamlit run app.py

**Step 6 — Open in browser**
http://localhost:8501

## 🙋 FAQ

**Q: Is this really free?**
A: Yes! OpenRouter gives free credits, Streamlit Cloud is free, SQLite is built into Python.

**Q: Do I need a credit card?**
A: No! Everything used in this project is 100% free with no credit card required.

**Q: How do I get the API key?**
A: Go to openrouter.ai — Sign up with Google — Keys — Create Key. Takes 2 minutes.

**Q: Will my past validations be saved?**
A: Yes! Every validation is saved in the sidebar under Past Validations so you can review them anytime.

**Q: Can I use this for any industry?**
A: Yes! The app supports 13 industries including EdTech, FinTech, HealthTech, AgriTech, E-Commerce and more.
